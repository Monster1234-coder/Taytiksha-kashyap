Smart Bin

The prototype model of Smart Bin:

Smart Bin is an automatic dustbin that collects waste lying near the bin and moves it inside.
Often, waste is scattered around dustbins, making the area unhygienic.
Smart Bin solves this problem by using **an ultrasonic sensor** to detect nearby waste.
The sensor sends signals to **an Arduino UNO**, which then commands **a servo motor** to rotate **a swiper arm**.
This arm moves **180¬∞** to push waste inside the bin, keeping the surroundings clean.
